##### USED

* Front
  * Bootstrap 4
  * Swiper
  * Scss
  * Express(ejs)
* Back
  * Node.js
  * Axios
  * MySql
  * Express-session

* Bundler
  * Gulp

##### Implement

* API data 출력
* 회원가입, 회원정보 수정
* 게시판 등록, 수정, 삭제
* 로그인 기록
